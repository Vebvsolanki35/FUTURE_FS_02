## 📌 About This Project

A personal professional portfolio website built from scratch using pure HTML, CSS and JavaScript — no frameworks, no templates. Designed to showcase my skills, projects, certifications and experience as a B.Tech CSE student at Lovely Professional University.

This is **Task 1** of the Future Interns Full Stack Web Development Internship Program.

## ✨ Features

| Feature | Details |

| 🎨 Dark Theme | Custom color palette with lime green accent |
